# nextjs-starter
A tool for creating `NextJS` project.


## Builtin
### Done
- Typescript
- Less
- Redux
- Antd
- TSLint
- StoryBook [branch/storybook](https://github.com/UtoYuri/nextjs-starter/tree/feature/storybook)

## Usage
### Dev Mode
```
yarn dev
```
### Production Mode
```
yarn build && yarn start
```
### Static Export
```
yarn export
```
### Lint
```
yarn lint
```
