# nextjs-starter
A tool for creating `NextJS` project.


## Builtin
### Done
- Typescript
- Less
- Redux
- Antd
- TSLint
- Prettier

## Usage
### Dev Mode
```
yarn dev
```
### Production Mode
```
yarn build && yarn start
```
### Static Export
```
yarn export
```
### Lint
```
yarn lint
```
### Format Code
```
yarn format
```
